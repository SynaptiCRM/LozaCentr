# LozaCentr — веб-каталог 2026

Ознайомча сторінка: фото, коди, розміри, ціни. **Кнопок «купити» немає.**

## PDF каталогу 2026

Кнопка **«Завантажити каталог PDF (2026)»** будує файл `LozaCentr-katalog-2026.pdf` **з поточних даних сайту** (`js/products-data.js`): ті самі назви, ціни, розміри та зображення, що на лендінгу. Це **не** статичний файл «Каталог LozaCentr 2024.pdf».

**Важливо:** відкривайте сайт через **локальний сервер** (Live Server, `python -m http.server`), а не подвійним кліком по `index.html`. Інакше браузер часто блокує малювання зовнішніх/локальних зображень у PDF (`file://`).

Потрібне інтернет-з’єднання для завантаження бібліотеки [html2pdf.js](https://github.com/eKoopmans/html2pdf.js) з CDN.

## Запуск

- **Live Server** у VS Code / Cursor на `index.html`
- або: `python -m http.server 8080` → `http://localhost:8080`

## Оновлення товарів і цін

Якщо у вас є `merge_products.py` у повній копії проєкту: змініть прайс у скрипті й виконайте `python merge_products.py`, щоб оновити `js/products-data.js`. PDF після цього автоматично відповідатиме новим даним.
